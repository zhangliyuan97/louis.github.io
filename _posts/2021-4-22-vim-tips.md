这篇文章用于记录自己在使用vim中的一些trick
* vim粘贴乱码，set pastetoggle=<F9>
* vim打开文件，光标定位到上次退出的位置
```
if has("autocmd")
  autocmd BufReadPost *
    \ if line("'\"") > 0 && line("'\"") <= line("$") |
    \   exe "normal g`\"" |
    \ endif
endif
```
